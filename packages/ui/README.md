# @camtomlabs/malix-ui

Malix Design System React components.

## Install

```bash
npm install @camtomlabs/malix-ui @camtomlabs/malix-tokens
```

## Usage

Import the token and component styles once:

```ts
import '@camtomlabs/malix-tokens/tokens.css';
import '@camtomlabs/malix-ui/styles.css';
```

Import components from the package:

```tsx
import { Button, Input, Card } from '@camtomlabs/malix-ui';
```

## Peer dependencies

- `react`
- `react-dom`

## Note

This package publishes raw TypeScript source. Configure your bundler to transpile the package in
frameworks like Next.js.
