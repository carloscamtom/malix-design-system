# @camtomlabs/malix-tokens

Malix Design System token package.

## Install

```bash
npm install @camtomlabs/malix-tokens
```

## Usage

Import the CSS variables globally:

```ts
import '@camtomlabs/malix-tokens/tokens.css';
```

Import the token registry when you need the allowed token list:

```ts
import registry from '@camtomlabs/malix-tokens';
```

## Included files

- `tokens.css`
- `tokens.registry.json`
- `index.js`
