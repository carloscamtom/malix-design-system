<ComponentPreview>

## Installation

```tsx
import { Avatar } from '@camtomlabs/malix-design-system';
```

## Usage

```tsx
<Avatar initials="CG" />
```

## Custom Size

Adjust the avatar size in pixels:

```tsx
<Avatar initials="CG" size={24} />
<Avatar initials="CG" size={40} />
<Avatar initials="CG" size={64} />
```

## Props

<PropsTable
  props={[

  ]}
