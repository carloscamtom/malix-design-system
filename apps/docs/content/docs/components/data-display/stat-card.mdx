## Installation

```tsx
import { StatCard } from '@camtomlabs/malix-design-system';
```

## Usage

```tsx
<StatCard
  label="Total Revenue"
  value="$48,250"
  change="+12.5%"
  changeType="positive"
/>
```

## Change Types

### Positive

```tsx
<StatCard label="Growth" value="24%" change="+4.2%" changeType="positive" />
```

### Negative

```tsx
<StatCard label="Churn" value="3.1%" change="-0.8%" changeType="negative" />
```

### Neutral

```tsx
<StatCard label="Active Users" value="1,204" change="0%" changeType="neutral" />
```

## Without Change

Display a stat without a change indicator:

```tsx
<StatCard label="Total Projects" value="42" />
```
