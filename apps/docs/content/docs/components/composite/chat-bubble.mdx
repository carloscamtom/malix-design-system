## Installation

```tsx
import { ChatBubble } from '@camtomlabs/malix-design-system';
```

## Usage

```tsx
<ChatBubble variant="ai" message="Hello from the AI!" />
<ChatBubble variant="user" message="User reply here" avatarInitials="CG" />
```

## AI Variant

The `ai` variant uses a light surface background with left-aligned rounded corners.

```tsx
<ChatBubble variant="ai" message="I can help with that." />
```

## User Variant

The `user` variant uses the primary CTA background, right-aligned with optional avatar initials.

```tsx
<ChatBubble variant="user" message="Thanks!" avatarInitials="JD" />
```

## Custom Content

Use the `children` prop for rich content instead of plain text.

```tsx
<ChatBubble variant="ai">
  <strong>Here are the results:</strong>
  <ul>
    <li>Component A</li>
    <li>Component B</li>
  </ul>
</ChatBubble>
```
