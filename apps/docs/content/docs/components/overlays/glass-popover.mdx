## Installation

```tsx
import { GlassPopover } from '@camtomlabs/malix-design-system';
```

## Usage

```tsx
<GlassPopover title="Quick Actions" onClose={() => setOpen(false)}>
  <p>Select an action to perform.</p>
</GlassPopover>
```

## Without Title

The popover can be rendered without a title for minimal layouts:

```tsx
<GlassPopover onClose={handleClose}>
  <ul>
    <li>Option A</li>
    <li>Option B</li>
  </ul>
</GlassPopover>
```
