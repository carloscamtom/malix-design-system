## Installation

```tsx
import { OnboardingPopover } from '@malix/ui';
```

## Usage

```tsx
<OnboardingPopover
  step={1}
  totalSteps={4}
  title="Create your first project"
  description="Click the + button in the sidebar to get started with a new project."
  onNext={() => setStep(2)}
  onSkip={() => setShowOnboarding(false)}
/>
```

## Custom Next Label

Override the next button label for the final step:

```tsx
<OnboardingPopover
  step={4}
  totalSteps={4}
  title="You're all set!"
  description="You've completed the onboarding. Start exploring the dashboard."
  onNext={handleFinish}
  nextLabel="Finish"
/>
```
