## Installation

```tsx
import { Modal } from '@malix/ui';
```

## Usage

```tsx
const [open, setOpen] = useState(false);

<Modal
  open={open}
  title="Confirm Action"
  onClose={() => setOpen(false)}
  onConfirm={() => {
    handleConfirm();
    setOpen(false);
  }}
>
  <p>Are you sure you want to proceed? This action cannot be undone.</p>
</Modal>
```

## With Custom Labels

Override the default button labels to match your context:

```tsx
<Modal
  open={open}
  title="Delete Project"
  onClose={() => setOpen(false)}
  onConfirm={handleDelete}
  confirmLabel="Delete"
  cancelLabel="Keep"
>
  <p>This will permanently delete the project and all its data.</p>
</Modal>
```
