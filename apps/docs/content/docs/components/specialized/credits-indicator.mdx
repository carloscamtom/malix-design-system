## Installation

```tsx
import { CreditsIndicator } from '@malix/ui';
```

## Usage

```tsx
<CreditsIndicator remaining={250} />
```

## Custom Label

Override the default label:

```tsx
<CreditsIndicator remaining={12} label="API Calls Left" />
```

## Low Credits Warning

The indicator can be styled contextually based on the remaining count:

```tsx
<CreditsIndicator remaining={3} label="Credits Remaining" />
```
