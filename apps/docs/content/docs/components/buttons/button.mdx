## Installation

```tsx
import { Button } from '@malix/ui';
```

## Usage

```tsx
<Button hierarchy="primary" variant="text">
  Create Project
</Button>
```

## Hierarchy Variants

The Button supports 4 hierarchy levels that control visual prominence:

```tsx
<Button hierarchy="primary">Primary</Button>
<Button hierarchy="secondary">Secondary</Button>
<Button hierarchy="tertiary">Tertiary</Button>
<Button hierarchy="ghost">Ghost</Button>
```

## Layout Variants

Four layout variants control the button content arrangement:

```tsx
// Text only (default)
<Button variant="text">Label</Button>

// Icon + Text
<Button variant="leading-icon-text" icon={<Plus size={16} />}>
  Create
</Button>

// Icon only
<Button variant="icon-only" icon={<Settings size={16} />} label="Settings" />

// Icon with badge
<Button variant="icon-badge" icon={<Bell size={16} />} badge="3">
  Notifications
</Button>
```

## Loading State

```tsx
<Button loading>Saving...</Button>
```
