Malix uses a closed set of color tokens defined as CSS custom properties. Every color in the system is intentional — no arbitrary hex values, no magic numbers.

## Usage

```css
.my-card {
  background: var(--malix-surface);
  color: var(--malix-foreground);
  border: 1px solid var(--malix-border);
}

.my-card:hover {
  background: var(--malix-surface-hover);
}
```
