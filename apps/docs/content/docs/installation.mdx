## Install the package

import { Tab, Tabs } from 'fumadocs-ui/components/tabs';

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab value="pnpm">
    ```bash
    pnpm add @camtomlabs/malix-design-system
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install @camtomlabs/malix-design-system
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @camtomlabs/malix-design-system
    ```
  </Tab>
</Tabs>

## Import the styles

Add the bundled Malix styles to your application entry point:

```tsx title="app/layout.tsx"
import '@camtomlabs/malix-design-system/styles.css';
```

Or import them in your global CSS file:

```css title="globals.css"
@import '@camtomlabs/malix-design-system/styles.css';
```

## Bundler Configuration

`@camtomlabs/malix-design-system` publishes raw TypeScript source, so configure your bundler to transpile the package.

### Next.js

```js title="next.config.mjs"
/** @type {import('next').NextConfig} */
const config = {
  transpilePackages: ['@camtomlabs/malix-design-system'],
};

export default config;
```

### Vite

```ts title="vite.config.ts"
import { resolve } from 'path';
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';

export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      '@camtomlabs/malix-design-system': resolve(__dirname, 'node_modules/@camtomlabs/malix-design-system/src'),
    },
  },
});
```

## Usage

Import components and start building:

```tsx
import { Button, Input, Card } from '@camtomlabs/malix-design-system';

export function MyComponent() {
  return (
    <Card title="Welcome" description="Get started with Malix">
      <Input placeholder="Enter your name" />
      <Button hierarchy="primary">Submit</Button>
    </Card>
  );
}
```

## Icon Library

Malix components accept any `React.ReactNode` for icon props. We recommend [Lucide React](https://lucide.dev/) for consistency with the design system:

```bash
pnpm add lucide-react
```

```tsx
import { Button } from '@camtomlabs/malix-design-system';
import { Plus } from 'lucide-react';

<Button hierarchy="primary" variant="leading-icon-text" icon={<Plus size={16} />}>
  Create Project
</Button>
```
