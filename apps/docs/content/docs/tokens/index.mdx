Malix uses a closed set of **CSS custom properties** organized into categories. These tokens are validated at build time — no ad-hoc values allowed.

## Usage

All tokens are available as CSS custom properties after importing the token stylesheet:

```css
@import '@camtomlabs/malix-design-system/tokens.css';
```

Use them in your styles:

```css
.my-element {
  background: var(--malix-surface);
  color: var(--malix-foreground);
  border-radius: var(--malix-radius-md);
  padding: var(--malix-space-md);
}
```
